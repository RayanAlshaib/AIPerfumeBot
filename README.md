#  AI Perfume Assistant (RAG System)

An intelligent perfume expert chatbot powered by RAG (Retrieval-Augmented Generation). This assistant provides accurate information about perfume brands, categories, and prices based on a curated dataset.

##  Features
- Smart Search:Arabic & English support with text normalization to handle various spelling variations.
- Custom Database: Utilizes `ChromaDB` as a vector store for efficient information retrieval.
- Advanced LLM: Powered by `Llama 3.3 70B` via `Groq API` for fast and human-like responses.
- Interactive UI: A sleek web interface built with `Streamlit`.

##  Tech Stack
- Language: Python 3.10+
- Orchestration: LangChain
- Vector Database: ChromaDB
- LLM: Groq (Llama-3.3-70b-versatile)
- Embeddings: HuggingFace Multilingual MiniLM
- Frontend: Streamlit

##  Project Structure
- `app.py`: The main entry point for the Streamlit web application.
- `ingest.py`: Script to process the CSV data and populate the vector database.
- `sample_data.csv`: The source dataset containing perfume details and pricingA `sample_data.csv` is provided for demonstration. For production use, replace it with your full dataset and update the file path in `ingest.py`.
- `chroma_db/`: Local directory storing the indexed vector embeddings (generated by `ingest.py`).
